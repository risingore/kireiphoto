# Code of Conduct

## Our Pledge

We as members, contributors, and maintainers pledge to make participation in this project a harassment-free experience for everyone, regardless of age, body size, visible or invisible disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

Examples of behavior that contributes to a positive environment:

- 他者への敬意を持った言動
- 建設的なフィードバック
- 自分の誤りを認め、学びに変える姿勢
- コミュニティ全体の利益を優先する判断

Examples of unacceptable behavior:

- 性的な言動 / 画像 / 接触
- トローリング、侮辱的・軽蔑的なコメント、人格攻撃 / 政治攻撃
- 公開・非公開を問わず、嫌がらせ
- 他者の個人情報 (物理アドレス、メールアドレス等) の本人同意のない開示
- プロフェッショナルな場で不適切と合理的に判断される行為

## Enforcement Responsibilities

プロジェクトメンテナーは適切で公正な是正措置を取る責任があります。不適切とみなされたコメント、コミット、コード、Wiki 編集、issue、その他の貢献を、本 Code of Conduct に違反するものとして編集 / 削除 / 拒否する権利と責任を持ちます。

## Scope

本 Code of Conduct は、プロジェクトの公式な space (リポジトリ、issue、PR、関連 Discord / Slack 等) と、メンテナーが代表者として行動する公開 space に適用されます。

## Enforcement

違反行為の報告は、リポジトリ owner (@risingore) まで連絡してください。すべての通報は速やかに公正に検討されます。報告者のプライバシーとセキュリティは保護されます。
